# Demo Script

## Goal
Show the complete workflow of the AI industry report system.

## Steps
1. Open the frontend demo page.
2. Input an industry topic and select a time range.
3. Submit a task to the backend API.
4. Observe the returned task identifier.
5. Open the report viewer page and query the task result.
6. Explain the agent workflow: planning, execution, analysis, memory, report generation.
7. Mention the DeerFlow adapter as the next integration point.

## Talking Points
- This is an AI agent oriented architecture.
- The system is ready for real data connectors.
- The platform can evolve into continuous industry monitoring.
